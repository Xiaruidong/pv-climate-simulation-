# 环境数据获取指南

## 📊 官方数据源清单

### 1. 环境数据 & 污染物排放量

#### 中国官方数据源
- **生态环境部**
  - 网址: https://www.mee.gov.cn/
  - 数据中心: https://www.mee.gov.cn/hjzl/
  - 内容: 环境统计年报、污染物排放数据

- **国家统计局**
  - 网址: http://www.stats.gov.cn/
  - 环境: http://www.stats.gov.cn/sj/hj/
  - 内容: 能源、环境统计年鉴

- **中国环境监测总站**
  - 网址: http://www.cnemc.cn/
  - 内容: 空气质量、水质监测数据

#### 国际数据源
- **世界银行**
  - 网址: https://data.worldbank.org/
  - 指标: CO2排放、能源消耗、污染排放

- **IEA (国际能源署)**
  - 网址: https://www.iea.org/data-and-statistics/
  - 内容: 能源统计、排放数据

- **OECD 数据库**
  - 网址: https://data.oecd.org/
  - 内容: 环境绩效指标

### 2. 光污染情况

#### 数据源
- **光污染监测数据**
  - 国际黑暗天空协会: https://www.darksky.org/
  - NASA地球观测: https://earthobservatory.nasa.gov/
  - 卫星遥感数据: DMSP-OLS、VIIRS夜光数据

#### 中国光污染研究
- 中国光污染地图研究
- 城市夜景亮度数据库
- 天文台观测数据

### 3. 节能量 & 能耗量

#### 数据源
- **国家能源局**
  - 网址: http://www.nea.gov.cn/
  - 内容: 能源消费、节能数据

- **中国标准化研究院**
  - 网址: http://www.cnis.gov.cn/
  - 内容: 节能标准、节能数据

- **节能产品惠民工程数据**
  - 工业节能数据
  - 建筑节能统计

### 4. 碳排放量

#### 中国官方数据
- **碳排放权交易数据**
  - 网址: https://www.cneeex.com/
  - 内容: 碳市场交易数据

- **中国碳核算数据库**
  - 网址: https://www.ceads.net.cn/
  - 内容: 详细碳排放数据

#### 国际数据
- **全球碳预算**
  - 网址: https://www.globalcarbonproject.org/
  - 内容: 全球碳排放数据

- **EDGAR排放数据库**
  - 网址: https://edgar.jrc.ec.europa.eu/
  - 内容: 全球温室气体排放

## 🔧 数据获取方法

### 方法1: 直接下载
1. 访问官方统计网站
2. 查找数据下载区域
3. 选择年份和指标
4. 下载Excel/CSV文件

### 方法2: API接口
- 世界银行API: https://datahelpdesk.worldbank.org/
- 环保部数据接口: 查看官方文档
- 第三方数据平台API

### 方法3: 爬虫获取
- 使用Python爬虫工具
- 定期抓取公开数据
- 遵守网站使用条款

## 📁 推荐数据集

### 高质量数据集推荐

1. **中国能源统计年鉴**
   - 覆盖: 全行业能源数据
   - 格式: PDF/Excel
   - 获取: 国家统计局官网

2. **中国环境统计年报**
   - 覆盖: 污染物排放、治理
   - 格式: PDF/Excel
   - 获取: 生态环境部官网

3. **CEADs碳核算数据库**
   - 覆盖: 详细碳排放数据
   - 格式: Excel/CSV
   - 获取: http://www.ceads.net.cn/

4. **世界银行发展指标**
   - 覆盖: 全球环境数据
   - 格式: CSV/API
   - 获取: https://data.worldbank.org/

## 🚀 快速获取建议

### 立即可用的数据源
1. 国家统计局 - 最权威
2. 生态环境部 - 最全面
3. CEADs数据库 - 最专业
4. 世界银行数据库 - 最便捷

### 数据获取优先级
1. **高优先级**: 中国官方统计数据
2. **中优先级**: 国际组织数据
3. **低优先级**: 研究机构数据

**注意**: 获取数据时请遵守相关网站的使用条款和数据版权规定。
