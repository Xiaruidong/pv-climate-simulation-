## 🔍 问题诊断

### 如果右侧没有更新：

#### 检查1：浏览器控制台
```
1. 按 F12 打开开发者工具
2. 查看 Console 标签
3. 寻找红色错误信息

常见错误:
- "undefined is not a function"
- "Cannot read property 'xxx' of undefined"
```

#### 检查2：参数范围
```
确认参数在合理范围内:
✓ 覆盖率: 1e-10 到 1e-6
✓ 反照率: 0 到 1
✓ CO2浓度: 280 到 1000
✓ 效率: 0 到 1
```

#### 检查3：组件状态
```
在浏览器控制台输入:
window.location.reload()

然后重新测试
```

### 结果解读指南

#### 温度变化
- **-1°C 到 -0.5°C**: 强降温效果（高反照率或高覆盖率）
- **-0.5°C 到 -0.1°C**: 明显降温效果（正常参数范围）
- **-0.1°C 到 0°C**: 轻微降温（低覆盖率）
- **0°C 到 +0.1°C**: 基本平衡或轻微升温

#### CO2减排
- **< 1 ppm**: 极低覆盖率，影响很小
- **1-10 ppm**: 低覆盖率，轻微影响
- **10-100 ppm**: 中等覆盖率，明显影响
- **> 100 ppm**: 高覆盖率，显著影响

## 🚨 异常情况处理

### 如果看到异常数值：

#### 温度变化 > 10°C
```
可能原因:
- 覆盖率设置过大（> 0.01）
- 反照率设置异常

解决方案:
1. 检查覆盖率是否在 1e-10 到 1e-6 范围内
2. 重置参数到默认值
3. 刷新页面
```

#### CO2减排为负数
```
可能原因:
- 计算逻辑错误
- 参数冲突

解决方案:
1. 检查所有参数是否在合理范围
2. 查看浏览器控制台错误信息
3. 重置参数并重新测试
```

#### 显示 "NaN" 或 "undefined"
```
可能原因:
- 初始加载未完成
- 某个参数未正确传递

解决方案:
1. 等待页面完全加载
2. 稍微调整任何参数触发重新计算
3. 如果问题持续，刷新页面
```
