# 光伏热效应模拟器 - 专业仪表板

## 🌟 项目概述

这是一个基于深色科技风UI设计的专业级光伏热效应模拟器仪表板，提供实时3D可视化、参数控制和性能监测功能。

## ✨ 核心特性

### 🎨 UI设计特色
- **深色科技风**: 专业深色主题，蓝色渐变高亮
- **响应式布局**: 三栏布局，适应不同屏幕尺寸
- **交互反馈**: 实时通知、错误提示、状态指示
- **专业配色**: 科幻感的青蓝色调，能量感的橙红色点缀

## 🎯 使用指南

### 基本操作流程

#### 1. 参数配置
1. 在左侧面板选择场景预设
2. 调整环境参数(反照率、云量、辐射等)
3. 滑块实时响应，数值即时更新
4. 参数验证自动提示错误

#### 2. 3D可视化交互
1. **旋转**: 鼠标左键拖拽
2. **缩放**: 鼠标滚轮滚动
3. **视角切换**: 点击3D/RD/Haan按钮
4. **温度查看**: 颜色标尺对照热图

#### 3. 实时监测
1. 右侧面板实时更新指标
2. 24小时趋势曲线展示
3. 系统状态持续监控
4. 错误提示即时反馈

### 参数说明

| 参数 | 范围 | 推荐值 | 说明 |
|------|------|--------|------|
| 反照率 | 0-1 | 0.35 | 表面反射率，>1无效 |
| 云量 | 0-100% | 35% | 影响太阳辐射强度 |
| 太阳辐射 | 200-1200 W/m² | 850 W/m² | 太阳辐射强度 |
| 温度 | -20~50°C | 32°C | 环境温度 |
| 风速 | 0-20 m/s | 5.5 m/s | 影响散热效果 |
| 湿度 | 0-100% | 25% | 相对湿度 |

## 🎨 设计系统

### 颜色规范
```css
/* 主色调 */
--color-blue: #4a9eff;      /* 主要蓝色 */
--color-cyan: #00d9ff;      /* 青色高亮 */
--color-green: #00ff9d;     /* 绿色成功 */
--color-orange: #ff9d00;    /* 橙色警告 */
--color-red: #ff4757;       /* 红色错误 */

/* 背景色 */
--bg-primary: #0a0e27;      /* 主背景 */
--bg-secondary: #131829;    /* 次级背景 */
--bg-tertiary: #1a1f38;     /* 三级背景 */
--bg-card: #1e2542;         /* 卡片背景 */
```

### 组件规范
- **圆角**: 4px(sm), 8px(md), 12px(lg), 16px(xl)
- **间距**: 4px(xs), 8px(sm), 16px(md), 24px(lg), 32px(xl)
- **阴影**: 多层次阴影系统，支持发光效果
- **动画**: 150ms(fast), 250ms(normal), 350ms(slow)

## 🔧 技术栈

### 前端框架
- **Vue.js 3**: 组合式API，响应式系统
- **Vite**: 现代化构建工具，快速热更新
- **Canvas API**: 3D场景渲染和图表绘制

### 样式系统
- **CSS Variables**: 设计令牌，主题一致性
- **Flexbox/Grid**: 响应式布局
- **CSS Animations**: 流畅过渡效果

### 浏览器API
- **Canvas 2D**: 图形渲染
- **RequestAnimationFrame**: 动画循环
- **Resize Observer**: 响应式调整

## 🛠️ 自定义开发

### 添加新的参数类型

1. 编辑`src/styles/dashboard.css`
2. 修改CSS变量值
3. 确保对比度和可访问性
4. 测试所有组件

### 扩展3D场景
1. 在`Visualization3D.vue`中添加新元素

4. 优化性能

## 🐛 故障排除

### 常见问题

#### 1. 3D场景不显示
- 检查浏览器Canvas支持
- 确认JavaScript无错误
- 尝试刷新页面

#### 2. 参数控制无响应
- 检查事件绑定
- 确认Vue响应式数据
- 查看浏览器控制台

#### 3. 样式显示异常
- 清除浏览器缓存
- 确认CSS文件加载
- 检查浏览器兼容性

### 性能优化

#### 降低3D渲染负担
- 减少光伏面板数量
- 降低渲染帧率
- 简化场景复杂度

#### 优化内存使用
- 及时清理动画
- 优化Canvas尺寸
- 避免内存泄漏
