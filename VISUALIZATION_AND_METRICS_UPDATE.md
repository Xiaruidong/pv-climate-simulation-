### 4️⃣ 新增功能

#### 覆盖率格式化函数

**智能显示逻辑**:
```javascript
const formatCoverage = (value) => {
  if (!value && value !== 0) return '---'
  const num = parseFloat(value)
  if (num >= 0.01) return `${(num * 100).toFixed(4)}%`
  if (num >= 0.001) return `${(num * 100).toFixed(5)}%`
  return `${(num * 100).toExponential(2)}`
}
```

**显示效果**:
- `1e-9` → `1.0000E-07%`
- `1e-8` → `1.0000E-06%`
- `1e-7` → `1.0000E-05%`
- `0.01` → `1.0000%`
