## 项目简介

这是一个基于零维能量平衡模型(0D-EBM)的光伏建设气候效应评估系统，通过可视化大屏展示光伏面板部署对全球气候的影响。

**系统特色**:
- **真实物理计算**: 使用国际标准CODATA 2018物理常数和IPCC AR5气候数据
- **高精度显示**: 所有结果显示保留4位小数
- **完整存储方案**: 支持localStorage本地存储和可选MySQL数据库
- **实时3D可视化**: 动态地球模型和参数面板联动

## 科学原理

### 核心方程

**能量平衡模型 (EBM)**:
```
dT/dt = (S × (1 - αₚ) - ε × σ × T⁴ + ΔF_CO2) / C
```

**行星反照率计算**:
```
αₚ = α_surface × (1 - coverage_ratio) + α_pv × coverage_ratio
```

**CO2辐射强迫**:
```
ΔF_CO2 = 5.35 × ln(C / C₀)
```

### 主要参数 (CODATA 2018 + IPCC AR5)

- **S**: 太阳辐射常数 (1361 W/m²)
- **σ**: 斯特藩-玻尔兹曼常数 (5.670374419e-8 W/m²/K⁴)
- **ε**: 发射率 (0.612)
- **C**: 地球系统热容量 (2.08×10⁸ J·℃⁻¹·m⁻²)
- **αₚ**: 行星反照率
- **C₀**: 参考CO2浓度 (280 ppm)

### 光伏效应

1. **反照率效应**: 
   - 高反照率面板增加行星反照率
   - 减少太阳辐射吸收
   - 计算公式: α_pv × coverage_ratio

2. **减排效应**: 
   - 光伏发电替代化石燃料
   - 减少CO2排放
   - 计算公式: PV_power × η_replacement × emission_factor

3. **双重效应**: 
   - 降温效应: 反照率增加 → 反射更多太阳辐射
   - 减排效应: CO2浓度降低 → 减少温室效应
